Soal no.1

Jelaskan cara kerja website serta jelaskan hubungan antara web server, domain, dan hosting

Jawaban:

  • Cara kerja website
    Saat seseorang mengunjungi sebuah website, permintaan (request) dikirimkan ke web server. Web server kemudian memproses permintaan tersebut dan mengirimkan kembali halaman web yang diminta kepada pengguna melalui internet.
  • Hubungan Antara Web Server, Domain, dan Hosting:
    • Saat seseorang mengetikkan alamat domain di browser (contohnya, www.nama_website.com), permintaan tersebut dikirim ke server DNS (Domain Name System) untuk mengonversi nama domain menjadi alamat IP. Setelah mendapatkan alamat IP, permintaan tersebut diteruskan ke web server yang memiliki file-file website terkait. Web server kemudian mengirimkan halaman web yang diminta kepada browser pengguna melalui internet. Hosting menyediakan tempat untuk menyimpan file-file tersebut sehingga web server dapat mengaksesnya dan menyampaikannya kepada pengguna.
